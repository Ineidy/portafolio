Proyectos Destacados

Sistema G&C de Inventario CampusLands

El Sistema G&C de Inventario CampusLands es una aplicación creada para gestionar eficazmente el inventario de activos en la sede principal de CampusLands en Bucaramanga. Esta herramienta permite administrar activos, personal y zonas dentro del campus, facilitando la asignación, el mantenimiento y el control de todos los elementos físicos y recursos humanos de la organización. Desarrollamos un sistema de gestión de inventario para Campuslands utilizando Python. Este proyecto optimiza el seguimiento y control de inventarios, proporcionando una interfaz intuitiva y funcionalidades avanzadas para administrar productos y recursos.

Space X

Implementamos un portafolio digital para una empresa aeroespacial, destacando servicios y capacidades de sus cohetes para el turismo intergaláctico. En colaboración con los equipos de diseño UI/UX y sistemas, desarrollamos una aplicación web en HTML, CSS y JavaScript que facilita la visualización de información técnica y misiones posibles.
